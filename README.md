# 💎 JewelryShop - Tỏa Sáng Phong Cách Của Bạn 💎

Chào mừng bạn đến với **JewelryShop**, ứng dụng mua sắm trang sức trực tuyến được thiết kế để mang lại trải nghiệm mua sắm tinh tế, tiện lợi và đẳng cấp ngay trên thiết bị di động của bạn. Dự án này là sự kết hợp giữa công nghệ hiện đại và niềm đam mê với vẻ đẹp của trang sức, nhằm tạo ra một không gian mua sắm ảo lấp lánh và đầy cảm hứng.

## ✨ Các Tính Năng Nổi Bật

- **Giao Diện Hiện Đại & Sang Trọng**: Thiết kế tối giản, tập trung vào việc tôn vinh vẻ đẹp của sản phẩm.
- **Khám Phá Bộ Sưu Tập**: Dễ dàng duyệt qua các bộ sưu tập trang sức theo từng chủ đề và phong cách riêng.
- **Tìm Kiếm Thông Minh**: Nhanh chóng tìm thấy món trang sức bạn yêu thích với bộ lọc chi tiết.
- **Giỏ Hàng & Thanh Toán (Đang phát triển)**: Quản lý các sản phẩm đã chọn và chuẩn bị cho các bước thanh toán an toàn.
- **Danh Sách Yêu Thích**: Lưu lại những sản phẩm bạn quan tâm để xem lại hoặc mua sau.
- **Tài Khoản Người Dùng**: Đăng ký, đăng nhập và quản lý thông tin cá nhân một cách bảo mật với Firebase.
- **Xây Dựng Bằng Công Nghệ Mới Nhất**: Sử dụng React Native và Expo, đảm bảo hiệu năng mượt mà trên cả hai nền tảng iOS và Android.

## 🚀 Công Nghệ Sử Dụng

- **Framework**: React Native & Expo
- **Ngôn Ngữ**: TypeScript
- **Quản Lý Trạng Thái**: React Context API
- **Routing**: Expo Router
- **Backend & Authentication**: Firebase
- **Styling**: StyleSheet API

## 🛠️ Hướng Dẫn Cài Đặt & Chạy Dự Án

1.  **Clone a repository về máy của bạn**:
    ```bash
    git clone https://github.com/cam1909/JewelryShop.git
    ```
2.  **Di chuyển vào thư mục `myApp`**:
    ```bash
    cd JewelryShop/myApp
    ```
3.  **Cài đặt các dependencies**:
    ```bash
    npm install
    ```
4.  **Khởi chạy ứng dụng**:
    ```bash
    npm start
    ```
    Sau đó, quét mã QR bằng ứng dụng Expo Go trên điện thoại của bạn.
